Ключевые возможности

Интеллектуальная маршрутизация

ML-модель автоматически определяет сложность запроса и выбирает оптимальную стратегию обработки

Многоуровневое кэширование

L1/L2/L3 кэш для мгновенных ответов на повторяющиеся и семантически похожие запросы

Отказоустойчивость

Каскадные fallback механизмы обеспечивают стабильную работу даже при сбоях отдельных компонентов

Гибкая настройка

Полный контроль над стратегиями, порогами и параметрами для точной настройки под ваши задачи

Простая интеграция

from hybrid_rag import HybridRAG

# Инициализация системы
rag = HybridRAG(
    language="ru",
    cache_enabled=True,
    monitoring_enabled=True
)

# Простой запрос (автоматически используется Classic RAG)
result = rag.query("Какая столица России?")
print(result.answer)  # "Москва"
print(result.strategy_used)  # "classic_rag"
print(result.latency_ms)  # 120

# Сложный запрос (автоматически используется Agentic RAG)
result = rag.query(
    "Проанализируй все изменения в коде за последнюю неделю "
    "и определи их влияние на производительность системы"
)
print(result.strategy_used)  # "agentic_rag"
print(result.agents_used)  # ["research", "code_analysis", "synthesis"]
